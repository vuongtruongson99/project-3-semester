## Technologies
Project is created with:
* Java: 11.0.10
* JavaFX: 8.0.251
* Scene Builder: 16.0.0
* MySQL Workbench: 8.0.23

## Setup
Clone the repository:
```
git clone https://github.com/vuongtruongson99/project-3-semester
```
Run the application using:
```
gradlew run
```
